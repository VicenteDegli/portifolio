Procurando um desenvolvedor fullStack

Vicente de Paulo Desafio, persistência e rsultado. Vamos desenvolver juntos, soluções para o seu negócio?!

Desenvolvedor web, apaixonado pelo mundo da programação. Capaz de desenvolver rapidamente novas habilidades e encara com prazer novos desafios! Tendo já desenvolvido trabalhos freelancer em Java para uma clinica de fisioterapia para cadastro e consulta de clientes e seus tratamentos. Também, já desenvolveu sistema para digitalização dos dados e fichas de idosos para um Asilo de Idosos, usando as linguagens, HTML, CSS, JavaScript e PHP. Atualmente se dedicando ao desenvolvimento web com as linguagens HTML, CSS e JavaScript e a tecnologias React, Node e Express. Ansioso em voltar a atuar na área com a qual mais se identifica!
